web 1, web2 ו web 3 שלושה הבדלים עיקריים בין

.הבדל ראשון: הרעיון הבסיסי

.אתרי האינטרנט היו מוגדרים כקריאה בלבד, כלומר המשתמשים לא יכלו לבצע אינטראקציה עם האתרים Web 1 -ב •
.מוגדר כ- קריאה-כתיבה, כך שהוא מאפשר את היכולת למשתמשים לתרום תוכן, וליצור אינטראקציה עם משתמשי אינטרנט אחרים Web 2 •
מאפשר למשתמשים גם לקרוא (לקלוט מידע) וגם לכתוב (להוסיף תוכן וליצור אינטראקציה) ,Web 2 כמו Web 3 •
יכול להמיר מידע שמקודם הוגדר לקריאה בלבד בעזרת מטא-דאטא למידע שניתן לאתר, להעריך ולמסור ע"י תוכנות מחשב, ובכך לתת אפשרות של אינטראקציה בין מכונות Web 3 אך בנוסף

הבדל שני: מאפייני השימוש

.משתמש ב-הייפר-לינקד וסימניות בדפים. אין תקשורת בין השרת למשתמשים. אתרי האינטרנט הינם סטטיים, מה שמאפשר רק גלישה בתוכן Web 1 •
.מאפשר אינטראקציה טובה יותר, בעזרת פונקציות ואפליקציות כמו הזרמת וידאו, מסמכים מקווים וכו'. דברים נהיו מקוונים ונשמרו על הסרברים Web 2 •
.הינו שילוב של טכנולוגיית אינטרנט וידע, המאפשר אפליקציות ופלטפורמות שמשתמשים יכולים לבצע אינטראקציה איתן, כמו מערכות המלצה אישיות Web 3 •

הבדל שלישי: מנועי חיפוש

.מנועי החיפוש מבצעים אחזור של תוכן מאקרו, המאוד מהיר אך יכול להיות לא מדויק או מסובך להבנה Web 1 -ב •
.מנועי החיפוש מאחזרים תגיד עם תוכן מיקרו. האחזור הינו מדויק יותר אך מאחר והתיוג הוא ידני מנועי החיפוש מכסים רק כמות קטנה של אתרים מכל הרשת Web 2 -ב •
.מנועי החיפוש מבצעים אחזור לתוכן מיקרו לפי טקסטים שתויגו באופן אוטומטי, מה שמאפשר תוצאה מדויקת וגם רחבה Web 3 -ב •


.דוגמא הממחישה את ההבדלים : חיפוש השאילתה "סרטים חדשים" במנוע חיפוש

.ניתן לצפות שנקבל מסמכים בעלי הטקסט עצמו שחיפשנו בשאילתה, והמסמכים יהיו סטטיים המציגים תוכן מסוים Web 1 -ב •
,נצפה שהתוצאה תהיה אתרים מוכרים בעלי תוכן על סרטים חדשים שיצאו, כמואתרים של בתי קולנוע. בנוסף נוכל לצפות כי תהיה אופציה של אינטראקציה של המשתמש עם האתר, Web 2 -ב •
.כמו ביצוע קנייה של כרטיס לסרט חדש בבית קולנוע
נוכל למצוא אתרים הרלוונטיים לשאילתה בדומה ל-2, אך נוכל למצוא גם אתרים שאינם מאוד מוכרים, כאשר בהם יש תוכן הרלוונטי ספציפית למשתמש שחיפש את השאילתה Web 3 -ב •
